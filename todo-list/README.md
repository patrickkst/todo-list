> APP NAME: todo-list
> Description:
> This is a a todo-list application being developed over the span of weeks as a project for code the dream react
> How to run:
> First install vite by using the command : npx create-vite@latest --template react
> npx is a command line interface allow us to download packages through the terminal.
> the command will download the package vite with the react template.
> This template will create and organized the needed files and folders for the application.

> [NOTE]
> to run the app use "npm run dev"
